# Block File

Block File is a desktop app for indexing large docx debate folders, browsing indexed directories like a file explorer, previewing extracted headers, and copying any header text in one click.

## Stack

- Rust backend (Tauri commands + SQLite FTS5 index)
- SolidJS frontend
- TailwindCSS styling

## Run

```bash
bun install
bun run tauri dev
```
